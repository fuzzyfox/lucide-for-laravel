# Lucide for Laravel

The domain language for a Laravel + Filament package that generates an always-current Lucide
icon set (blade-icons + a `ScalableIcon` enum) from the official `lucide-icons/lucide`
repository, with optional Filament icon-alias overrides.

> **Scaffold only.** This glossary is intentionally empty. Terms get added during the design
> grilling session (`/grill-with-docs`), one at a time, as each is resolved and made
> canonical. Until then, treat the `README.md` idea dump — not this file — as the source of
> early language.
>
> Keep this file a **glossary and nothing else**: define what each term _is_ in one or two
> sentences, pick one canonical word per concept and list the aliases to avoid, and keep all
> implementation detail out. See the producer skill's `CONTEXT-FORMAT.md` for the format.
